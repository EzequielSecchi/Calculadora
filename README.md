# Calculadora
Este es mi primero proyecto en HTML, CSS y JavaScript. Aqui decidi crear una calculadora basica y, con el paso el tiempo, irla actualizando agregandole cosas.

Estas son un par de modalidades puestas recientemente en este proyecto:

## Calculos basicos
Fue implementada la suma, resta, multiplicacion y division, tanto con numeros enteros como con decimales. Se producen errores en caso de que un calculo sea erroneo (por ejemplo,  una division por cero)

## Implementacion del Ans
El dato resuelto anteriormente queda guardado temporalmente en memoria hasta que se resuelva otro calculo. Cuando se quiera volver a utilizar un resultado para otra ecuacion, se utiliza el Ans para colocar en el visor el valor del resultado

## Implementacion de DEL y AC
Con el boton AC, se borra el resultado en el visor. Con el boton DEL, se borra solo el ultimo numero o simbolo puesto.

## Implementacion del modo oscuro
Se agrego un boton que permite poner el fondo en modo oscuro o claro, dependiendo las preferencias del usuario
